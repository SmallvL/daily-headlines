设计要点

  • 布局：卡片水平垂直居中，四周留白。结构改动最小。
  • 卡片：大圆角（20px）、柔和阴影、纯白背景；自定义背景时卡片带轻微透明 + 毛玻璃效果。
  • 品牌：渐变圆角 Logo 图标 + 应用名 + slogan。
  • 表单：圆角输入框，主色登录按钮，错误提示置顶。
  • 动效：Framer Motion 卡片入场、聚焦过渡、按钮悬停微缩放。
  • 默认背景：浅灰渐变（不跟随暗色主题）。
  • 自定义背景：支持上传图片，登录页用 background-image: url(...) 覆盖，卡片透明度自适应。

新增配置项

  • 设置页新增："登录页背景" 设置卡片
  • 选项：使用默认 / 上传自定义图片 / 清除自定义图片
  • 后端：扩展 preferences 模块，支持保存 login_background_url
  • 存储：图片上传复用现有图片代理/静态文件机制，或新增上传接口

技术方案

  • 新增依赖：framer-motion
  • 前端：LoginPage.tsx 读取 login_background_url 偏好，动态切换背景
  • 前端：SettingsPage.tsx 新增背景图上传/清除 UI
  • 后端：扩展 preferences API 与数据库字段
  • 样式：登录页专用样式集中在组件内
