## 三、API 格式兼容性测试

### 3.1 OpenAI 格式 (mimo-v2.5)

```
端点: https://token-plan-cn.xiaomimimo.com/v1/chat/completions
认证: Authorization: Bearer <key>
请求: messages + model + temperature + max_tokens
响应: choices[0].message.content + usage
```

**结果:** ✅ 完全兼容，返回正常 JSON

### 3.2 Anthropic 格式

```
端点: https://token-plan-cn.xiaomimimo.com/anthropic/messages
认证: x-api-key + anthropic-version
请求: system (top-level) + messages + max_tokens
响应: content[{type:"text", text:"..."}] + usage.input_tokens/output_tokens
```

**结果:** ⚠️ MiMo 平台不支持 Anthropic 格式（返回 404）
**代码状态:** ✅ 实现完整，待用真实 Anthropic API 验证
