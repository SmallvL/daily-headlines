# 登录页视觉优化设计文档

**日期：** 2026-06-16  
**范围：** `apps/web/src/modules/auth/LoginPage.tsx` 及其相关依赖  
**目标：** 提升登录页品牌感与视觉质感，同时支持用户在设置页自定义登录背景图。

---

## 背景与目标

当前登录页为简单的居中表单卡片，视觉表现较为朴素。本次优化希望：

1. 在不大改布局结构的前提下，提升登录页的品牌质感。
2. 采用极简专业风格，与后续全局导航的混合式布局相协调。
3. 为登录页背景提供可配置能力，默认使用品牌渐变，用户可在设置页上传自定义背景图。

---

## 最终方案

采用 **方案 A：悬浮居中卡片 + 可配置背景**。

### 视觉设计

#### 桌面端

- 页面背景默认使用浅灰渐变（`linear-gradient(135deg, #f5f7fa 0%, #e4e7f1 100%)`）。
- 登录卡片水平垂直居中，宽度约 `360px`，内边距 `40px 32px`。
- 卡片使用大圆角（`border-radius: 20px`）、柔和弥散阴影（`box-shadow: 0 20px 60px rgba(0,0,0,0.08)`）。
- 顶部放置渐变圆角 Logo 图标、应用名"我的每日头条"、一句话 slogan"汇聚你关心的内容"。
- 表单区域使用圆角输入框（`border-radius: 10px`），主色登录按钮。
- 错误提示以 Alert 形式置顶在卡片内部。

#### 自定义背景图

- 当用户在设置页上传背景图后，登录页背景替换为自定义图片（`background-image: url(...)`），默认 `center/cover`。
- 卡片在自定义背景模式下使用半透明白色背景（`rgba(255,255,255,0.96)`）并添加轻微毛玻璃效果（`backdrop-filter: blur(8px)`），确保表单可读。
- 阴影加深（`0 20px 60px rgba(0,0,0,0.12)`），增强卡片与背景的层次分离。

#### 移动端

- 卡片宽度自适应，最大宽度 `360px`，水平内边距缩小为 `24px`。
- 背景保持全屏，卡片仍居中显示。

### 动效

- 卡片入场：使用 `framer-motion` 实现轻微上浮淡入（`y: 20 → 0`, `opacity: 0 → 1`）。
- 输入框聚焦：边框颜色过渡。
- 按钮悬停：轻微缩放（`scale: 1.02`）与阴影加深。

### 主题策略

- 登录页不跟随应用的暗色/亮色主题设置，始终保持品牌渐变或自定义背景。
- 进入应用内部后，再由设置页的主题偏好控制整体风格。

---

## 数据流与配置

### 新增用户偏好字段

扩展 `UserPreference` 类型，新增字段：

```ts
login_background_url?: string | null;
```

- `undefined` / `null`：使用默认渐变背景。
- 非空字符串：使用自定义图片 URL 作为背景。

### 设置页交互

在 `SettingsPage.tsx` 新增"登录页背景"配置卡片：

- 当前为默认渐变时，显示默认状态与"上传背景图"按钮。
- 已上传自定义背景时，显示图片缩略图与"清除"按钮。
- 上传成功后更新偏好并回显。

### 后端扩展

- 扩展 `preferences` 模块的数据模型与 API，支持保存/读取 `login_background_url`。
- 图片上传复用现有静态文件服务或新增独立上传接口；URL 可为相对路径或绝对路径。

## 技术约束

- 继续使用 Ant Design 的 `Form`、`Input`、`Button`、`Alert` 组件，保持表单验证逻辑不变。
- 登录页样式优先使用组件内样式（styled/inline），避免向全局 `styles.css` 增加大量登录页专用规则。
- 新增依赖 `framer-motion` 仅用于登录页入场动效，后续如需更多动效可复用。

---

## 验收标准

- [ ] 桌面端登录页显示悬浮居中卡片，包含 Logo、应用名、slogan、表单。
- [ ] 默认背景为浅灰渐变。
- [ ] 输入框聚焦、按钮悬停有平滑过渡动效。
- [ ] 卡片入场有轻微上浮淡入动效。
- [ ] 移动端卡片自适应，内容可读。
- [ ] 设置页可上传自定义背景图，上传后登录页背景立即生效。
- [ ] 设置页可清除自定义背景图，清除后恢复默认渐变。
- [ ] 自定义背景图时，卡片保持可读性（半透明 + 毛玻璃）。
- [ ] 现有登录 API 与认证流程不受任何影响。
